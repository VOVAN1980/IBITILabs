# Contributing to IBITI Labs

Thanks for your interest in contributing.

This repository is the **umbrella landing + pointers** for IBITI Labs.
Core protocol development happens in dedicated repos (e.g. EPK).

## Ways to contribute

- Fix typos / improve docs
- Improve the umbrella landing page (index.html)
- Propose architecture notes, diagrams, or clarifications
- Open issues with clear reproduction steps / suggestions

## How to propose changes

1. Fork the repo
2. Create a branch: `feat/<name>` or `fix/<name>`
3. Keep changes focused and small
4. Open a Pull Request with:
   - what changed
   - why it matters
   - screenshots if UI changes

## Style

- Prefer clarity over marketing
- Avoid hype words, keep claims verifiable
- Keep links canonical and up to date

## Code of Conduct

By participating, you agree to follow the Code of Conduct.
