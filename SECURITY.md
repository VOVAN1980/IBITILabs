# Security Policy

IBITI Labs takes security seriously.

## Reporting a vulnerability

Please report security issues **privately**.

- Email: **info@ibiticoin.com**
- Subject: **[SECURITY] <short title>**
- Include:
  - affected repository / component
  - steps to reproduce (minimal)
  - expected vs actual behavior
  - potential impact
  - any suggested fix (optional)

## Disclosure process

We will:
1. Acknowledge receipt within **72 hours**
2. Triage and validate the report
3. Coordinate a fix and release
4. Publish a brief advisory when appropriate

Please do **not** open public GitHub issues for vulnerabilities.
